The premise

The pull request is the atomic unit — not the file.

AI made writing code fast, which moved the bottleneck to review. Queues grow, PRs sit for days, and you burn time context-switching: open GitHub, find the PR, read the diff, check CI, read comments, switch tab, repeat. CodeLobby collapses that loop into one view and asks the questions that actually matter — what does this PR solve, does CI pass, what's blocking merge?

Everything, in one lobby

Built for reviewing at speed

A spatial home for your pull requests, with the full context and the actions right where you need them.

Spatial PR canvas

Every repo and its open PRs as draggable, resizable cards on an infinite grid. Your layout, your way — and it persists.

IDE tree view

Prefer a tree? Flip to a familiar repository → PR explorer with keyboard navigation, grouped by author.

Full PR context

CI checks, comments, reviews, threads, labels, commit history and complete file diffs — no tab-switching.

Act from the app

Approve, merge (squash/merge/rebase), comment, resolve threads, update branch, manage labels — one click.

CI at a glance

Every check run, grouped by state with search and filters, live-updating — click through to the logs on GitHub.

Network & rate-limit monitor

A built-in inspector for every API call — method, timing, payloads, GraphQL queries — plus a live GitHub rate-limit gauge.

Claude, built in

An AI reviewer that reads the whole PR.

CodeLobby integrates Claude with real tools — it can read files, search the codebase, run git, and fetch the web to actually understand a change before it says a word. Powered by Claude Code and your own Claude subscription.

  • “Why is this PR still open?” — one click surfaces every blocker: failing CI, pending reviews, unresolved threads, draft state.
  • AI-generated reviews — Claude drafts a structured review you edit and submit to GitHub with inline comments.
  • CI-failure diagnosis — it pulls the logs, finds the root cause, and suggests a fix.
  • Per-PR chat & quick actions — full PR context, streaming reasoning, and your own saved prompts.

How it works

01

Connect GitHub

Paste a personal access token with repo scope. It's stored locally & encrypted — your data never leaves your machine.

02

Pick your repos

Choose the repositories to watch. CodeLobby pulls every open PR with its CI, reviews and comments into one canvas.

03

Review & ship

Read, ask Claude, and act — approve, comment, or merge — without ever leaving the app.
